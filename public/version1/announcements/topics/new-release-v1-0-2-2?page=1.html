Hello everyone,

We've just pushed a new version of GestureWorks Gameplay to the server. Owners of Gameplay will receive an update notification within the application (or you can click "Check for updates" in the Info dialog) and there is also a new trial version installer available for download on the Gameplay website.

This is a maintenance release that fixes some issues that appeared in v1.0.1.2 with regard to the launching of Steam games. In some instances, some games that previously worked would either fail to launch or would launch without an overly. This new version includes a more robust launch process that should alleviate these problems.

As always, please post any questions to the forums, and if you think you've found a bug, please feel free to submit a bug report:

https://gameplay.gestureworks.com/bug_reports/new/

Thanks for using Gameplay!

--Marsh
