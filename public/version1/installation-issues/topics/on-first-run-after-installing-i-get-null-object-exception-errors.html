I think their are a few possibilities, blairhacche.

Do you have a password for your Windows account? I think what is most likely the problem is that your Ideum file is not being placed into the Program Data file due to an odd bug caused by not having a password for your Windows 8 account. In that case, you could simply uninstall Gameplay, make a Windows 8 password, re-install, and see if that works. An upcoming build should fix it soon too though.

Another issue that seems to be popping up a lot has to do with a default Windows 8 setting that causes restart to do a "hybrid restart", meaning it doesn't fully restart your computer like is needed to run the tool. You might need to go into your settings to change this.

Let me know!
