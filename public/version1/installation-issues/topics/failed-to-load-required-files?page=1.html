The version that you have may not be the latest, and the fix for this is to do a clean install, which is outlined here: https://gameplay.gestureworks.com/gameplay_forums/forums/installation-issues/topics/how-to-clean-install-of-gwg

Once you are done with step 3, make sure you download the new version of Gameplay from this link: https://gameplay.gestureworks.com/system/installers/GestureWorksGameplaySetup_0.9.0.0.exe

Also, remember that you'll need to restart your device twice -- once after installing the program, and a second time after installing the updates. It's important that it restarts that second time.

Hope this helps! Let me know if the error message persists. Thanks!
