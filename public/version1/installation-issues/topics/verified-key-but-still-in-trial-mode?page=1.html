Hi, bluedory! Thanks for your question on this. It sounds like this is a bug, as you did everything correctly. The only thing I might suggest that may help is to do a "clean install." To do this, follow these steps.

  1. First, make sure any game clients you may have (such as Steam or Origin) are closed. Press the Windows button + R on your keyboard to bring up the Run command. Type in %appdata% and delete the Ideum folder entirely. Press Windows + R again and this time type in %programdata% and delete the Ideum folder in there.

  2. Press Windows + R and type: regedit Hit Yes to the pop-up box. HOWEVER! Please be extremely careful not to delete anything else other than what follows, otherwise your computer may be seriously damanged. First, expand/double click on HKEY_CURRENT_USER -> Software -> Ideum -> GestureWorks -> Gameplay. Then delete your LicenseKey from that list. (Make sure you have it written down or saved somewhere.)

  3. Open the Charms Bar to go to your Settings and Control Panel. Under Programs and Features, Uninstall GestureWorks Gameplay.

  4. Reinstall GestureWorks Gameplay (you should still have the link to the download in your Beta Invite e-mail) and restart.

  5. Enter in your License Key when prompted and install the updates.

  6. After the updates complete, restart your computer.

If this process still does not work and the Trial Mode issue persists, please submit this as a bug using our Bug Report: https://gameplay.gestureworks.com/bug_reports/new/

Thank you very much!
