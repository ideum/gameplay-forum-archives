Okay, a few possibilities, Remmcraw.

If you are using Windows 8.1, a default set-up causes restart to do a "hybrid restart", meaning it doesn't fully restart your computer like is needed to run the tool. You must go into your settings to change this. I think this is the most likely problem.

Alternatively, it could be that your Ideum file is not being placed into the Program Data file due to an odd bug caused by not having a password for your Windows 8 account. In that case, you could simply uninstall Gameplay, make a Windows 8 password, reinstall, and see if that works.

Let me know!
