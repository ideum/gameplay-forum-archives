FIX

Before you start the game.

In "c:\programdata\ideum\gestureworks\gameplay\live\images\keyboard"

Replace "keyboard_down.png" and "keyboard_up.png" with "blank_key1_down.png"

Then run the game.

Don't know why GWG isn't loading those 2 files properly (at least on my hardware) but I have alerted the devs.

This is a non-permanent solution, you can make it sort of permanent if you make the switch in program files as well.
