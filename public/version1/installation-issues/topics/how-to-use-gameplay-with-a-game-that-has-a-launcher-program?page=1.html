Hi Adrian!

I'm not a staff member here, I'm just another user of gestureworks. Normally I'd let one of the pro's handle this, but 10 days have gone by, they must be very busy with gestureworks 2.0 (looks fantastic!).

I have no issues playing games that have a launcher, I'll use oblivion for example. I bind gestureworks to the actual game executable (not the launcher) and launch the game as normal from steam. Steam runs the launcher, i click 'play' and let it run and BOOM, I've got my UI overlaid.

Now... to be completely honest: to say "I have no issues playing X" with gestureworks is almost a lie. Gestureworks isn't a bad piece of software, it must be difficult to make something that will work with all these games. A caveat of gestureworks I've run into on my setup is needing to reboot and relaunch everything when my UI fails to show... I've gotten fairly used to it and at first I thought my install might be broken, etc, but no, sometimes a reboot is all I need to get things running again. It's odd, but I've learned to live with it and hope an update will iron these things out.

Good luck and be sure to reply if you need any additional help!
