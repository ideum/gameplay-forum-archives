Hi qwasz123, the behaviour you're describing can be set up with Global Gestures. Try adding a 1-finger Drag gesture with output set to Left Mouse Relative; for games that use the mouse to move the camera, this should allow you to move the camera with finger swipes.

We haven't generally recommended this yet because it will also issue whatever command is set for left-click, which (in first-person shooters) is commonly firing a weapon. You can circumvent that by reassigning whatever left-mouse does to something else. We realise it's a bit of a sub-optimal solution, and we're working on adding a MouseLook option for drag gestures in a near-future update, so stay tuned!
