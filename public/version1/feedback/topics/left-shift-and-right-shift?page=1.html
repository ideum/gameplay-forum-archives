Is there a way to differentiates the left and right shift keys? When I create mappings for a new game, I can only assign "Shift" to a button. There's no difference between left or right shift.

The original controls use left and right shift keys to activate the flippers, so it's essential to me that I can have a button for the left shift key and one for the right shift key. I can assign different keys in the game and map the keys to those, but then again, while it works well using the Gameplay software, I need to change back the keys to the default whenever I play with the keyboard. But otherwise, a very nice piece of software.

Some games do require the use of both the left and right shift or ctrl keys for different actions, but Gameplay is not able to differentiate between the two. The way to work around this would just been to customize the controls in game to be something else, and then you can assign those new controls to Gameplay buttons. Hope this helps!
