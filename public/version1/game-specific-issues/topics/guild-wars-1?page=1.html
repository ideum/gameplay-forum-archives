Hi Team, Another issue, this time with Guild Wars 1 (which I haven't played for sometime, and now have a little bit of spare time to test the game with GWG).

So, as I did when testing Guild Wars 2, I've created a simple Layout with just a D-Pad assigned to W,S,A,D (Up, Down, Left, Right)

I Start the game, the first screen presented is the logon screen. When I press the buttons on the Virtual D-Pad I can see in the username box the letters W,S,A,D being input. This tells me that the Virtual controller is working. I then use the virtual Keyboard built into GWG to enter my email and password and login Guild Wars.

After i select a character and wait for it to load up, I then try to use the Virtual D-Pad to move the character, Alas the character does not move.

Feeling a little disappointed i had another thought and setup another Virtual Controller. This time I set a Global Gesture for "Drag" and assigned Mouse Look, with Right Click as the modifier. (In Guild Wars, Holding the RIght Mouse button and moving the mouse simulates Mouse Look).

Again i load up a character and low and behold, The Drag Gesture Works! i drag my finger around the screen and the camera moves just as expected. As another test, i added both Left and Right mouse buttons as the modifier`to mouse look, The expected result is that the character should both move forward and be navigable. And Yes! it works. I then assigned Mouse Clicks, etc to Virtual Buttons, these all work.

It seems that actions that uses keyboard input doesn't work for this game.

I hope there is enough information here for you guys to use. As ive stated before this software is incredible and i think also very underrated (Though it is great to here you guys got Greenlit on steam, congrats!). I am also trying to get the Debug and Log files sent aswell.
