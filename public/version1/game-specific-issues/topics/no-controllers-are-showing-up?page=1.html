acousticguru
9 months ago
  - In reply to maximillyan.lambert
 

No worries! We will have you smashing zombies in no time :)

First make sure you downloaded NMRIH from steam.

Next create a new Virtual Controller or use the Half Life Virtual Controller that's already shared.

Go into the Virtual Controller's edit screen then click on the Settings tab.

Assuming you installed Steam in the default location, copy this path into the Executable Path field: C:\Program Files (x86)\Steam\steamapps\common\nmrih\sdk\hl2.exe

Gameplay lets you pass command line arguments and you have to pass a couple in order to play NMRIH. While still in your Settings tab of the Virtual Controller, copy this into your Command Line Arguments field: -game ..\nmrih

Next click the green launch button in the upper right of the tool.

Final step, survive hordes of zombies. Good luck and be sure to share your Virtual Controller if you make one!
