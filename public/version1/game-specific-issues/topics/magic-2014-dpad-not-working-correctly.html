I talked with one of the devs about this who checked it out as well. He believes that in this version the game is processing touch events natively which is making the d-pad misbehave. A way to work around this could be to add a global gesture for mapping zoom to the mouse wheel, flipping through the deck, etc. instead of the d-pad. Hope this helps!
