Hi lowman11,

Gameplay should work in either windowed mode or fullscreen. I just doubled checked that FFVII worked in both with a Virtual Controller overlay. The one thing I would suggest is always linking your Controller to the Launcher EXE (if there is one). So, in this case, use FF7_Launcher.exe. Typically, you want to have the EXE pointing to the launcher, but even still, I am able to get an overlay when having the executable path point toward ff7_en.exe.

If this problem persists, and restarting your device doesn't seem to help, please make sure you've installed the latest update to Gameplay, as this was a pretty crucial update to getting overlays to appear as they should in games, and if that still does not help, could you submit a bug report? https://gameplay.gestureworks.com/bug_reports/new/

Thanks!
