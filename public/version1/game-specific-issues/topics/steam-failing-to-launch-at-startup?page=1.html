Thanks for sending your logs. It looks like the reason that Gameplay is failing to launch Steam at startup is that for some reason your Steam executable is set to run as administrator only. This would explain why you were able to launch it from the command prompt (assuming command prompt was also launched as admin).

You can fix this by using Windows Explorer to navigate to your Steam executable. The default location (assuming your primary drive is "C") is:

32-bit Windows: C:\Program Files\Steam\Steam.exe

64-bit Windows: C:\Program Files (x86)\Steam\Steam.exe

Right click on Steam.exe, open the properties menu and click on the Compatibility tab. Near the bottom of the menu you should see a check box that says "Run this program as an administrator". Make sure this is unchecked and click Ok.

Next time you log in to your Windows account, Gameplay should launch Steam automatically.

NOTE: This will only fix the startup issue, if you still get the red "Failed to load" message in-game let us know and send your logs right after you get this message.
