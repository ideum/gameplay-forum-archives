Hello all,

I'm having a very specific issue with some games on my Transformer Book T100. It comes with a keyboard that shows "HID Keyboard Device" under keyboards and "Asus touchpad" under mice and other pointing devices in Device Manager. These only show when the keyboard is attached, and become hidden when not attached.

The problem I've run into is that I must have the detachable keyboard that comes with the tablet attached when I start games (and sometimes I must navigate the menus to start the actual gameplay with it attached) or else none of the mouse gestures work properly. (They either don't move at all or they don't move the in game mouse as they should)

For example, if I start Oblivion with the keyboard in, the downloaded controllers from the gestureworks database work perfectly. They operate the mouse and do everything else. I can even detach the keyboard once started and they will still work. If I start Oblivion without the keyboard attached, however, the mouse movement inputs (regardless of whether I have them assigned to the joystick or to the touch screen) don't work. It is as if the keyboard/touchpad must be attached for the gestureworks on screen mouse buttons/mouse gestures to work.

Same thing happens in Tiger Woods 08. As long as the keyboard is attached when I start the game and the round, I can remove the keyboard and everything will work using just the tablet. But if I start the game without it attached, the mouse buttons/touchscreen mouse controls won't work once I begin the swing animation (they do work prior to it).

I have tried everything I could think. As I said, when I detach my keyboard, the keyboard and touchpad devices become hidden in device manager, as they are supposed to. I was thinking if there was some way I could trick the tablet into thinking the physical keyboard/touchpad were attached everything would work, but I don't know how to do that.

I've also tried using an onscreen mouse emulator (TouchMousePointer) in various ways, and while this will sometimes give me control of the pointer depending on the game, it doesn't work as desired in Oblivion or TW08.

Can anyone think of a workaround for this? I almost never have the keyboard attached and having to plug it in every time I want to start the game (or every single round of golf in TW08) defeats the purpose of having a mobile tablet that plays games using only the screen. If I wanted to use a keyboard every time I played, I'd just use my other laptop.

Any thoughts?

 
jtr110x
3 months ago
 

Thought I would let everyone know my progress in case anyone else had the issue.

I found a workaround that will "start" my touchscreen mouse controls without the keyboard having to be attached. Basically, I found that if I ever got the mouse controls in gestureworks working (either touch gestures or joystick) -- which happened seemingly randomly every now and then but appears to have been related to the way mousekeys interacted with my physical keyboard -- they would stop working once the physical keyboard was attached and then detached. I then could not get them working again without reattaching my physical keyboard.

The workaround to get them restarted was to go into mousekeys, uncheck or check any setting, and then click apply(of course mousekeys must be activated). This brings mouse control back in gestureworks without having to connect the physical keyboard.

I'll keep looking for a more elegant solution, but this works for now as it allows me to start and play games without the physical keyboard ever attached. In fact, if I also keep touchmousepointer open while tiger woods 08 is running, it pretty much runs perfectly, though the on screen buttons still don't work mid swing.

Anyways, thought I'd let people know just in case anyone was having the same problem.

Hi decon1313,

It doesn't sound like you're doing anything wrong. While many games are compatible with Gameplay, you may come across some which are not. We've tested both Jamestown and Gunpoint and found that they are not compatible. Gameplay is currently only compatible with DirectX 9c or later games. If you launch a game and the home button for Gameplay isn't showing up on the side of the screen then you will know it is not compatible.

Thanks for bringing this up as some other users may have come across these games and wondered about them as well.
