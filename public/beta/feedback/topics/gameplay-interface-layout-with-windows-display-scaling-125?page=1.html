Hi,

First of all, thanks for allowing me to take part in the beta testing!

I've just done a clean uninstall of v1 from Steam, followed by install of the current beta. All seems to have gone well.

I've taken a quick look around the interface, and am happy to see that a number of the niggles I had with v1 have already been taken care of!

Only one extremely nit-picky aesthetic thing I've spotted so far...

I'm testing on an Asus T100, which has a native resolution of 1368x768 (according to my device, although I've seen specs online list 1366x768). However, I have Windows set to do 125% display scaling, which generally makes things easier to tap with fingers on a touch-screen!

Good news... all of Gameplay's interface seems to cope with this perfectly fine!

The only tiny issue with this is when you go into 'edit' mode on a virtual controller, the row of buttons along the top has a tiny amount of text cut off the left-hand side of each one (not a big problem, there's enough showing to see what they are for!). Perhaps the text on the buttons could be left-aligned to the button edges, for readability. Even better, maybe auto-size the text to fit?

Also, when I hit the 'Properties' button, the list down the right-hand side was hanging off the edge a tiny bit... although once I minimised & then maximised the app, it seems to have permanently sorted itself out, so not sure what happened there.

I've got a screen-grab, but not sure how to attach it here. Hopefully my explanation is enough to picture it anyway.

Told you it was only a tiny thing! So... looking good so far. I'll report back once I've given it a more thorough test, especially on some of the games I couldn't get to work with v1.
